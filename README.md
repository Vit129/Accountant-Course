# เตรียมสอบ CPA — Accountant Learning Portal

เว็บเรียนแบบ interactive สำหรับเตรียมสอบ **ผู้สอบบัญชีรับอนุญาต (CPA)** ของไทย ครอบคลุม 6 วิชา พร้อมข้อสอบจำลองแบบจับเวลา

🔗 **Live site:** https://vit129.github.io/Accountant-Course/

## เนื้อหา

| วิชา | หัวข้อ |
|---|---|
| 1. การบัญชีการเงิน 1 | [`01-financial-accounting-1`](01-financial-accounting-1/) |
| 2. การบัญชีการเงิน 2 | [`02-financial-accounting-2`](02-financial-accounting-2/) |
| 3. การสอบบัญชี 1 | [`03-auditing-1`](03-auditing-1/) |

แต่ละวิชามีบทเรียนย่อยพร้อมแบบทดสอบ และมี[ข้อสอบจำลองแบบผสม จับเวลา](exam/)รวมทุกวิชา

## ข้อจำกัด (ตรงไปตรงมา)

ข้อสอบจำลองในเว็บนี้เป็น**ตัวแทนหัวข้อ** ไม่ใช่ข้อสอบเก่าจากสภาวิชาชีพบัญชี และไม่ครอบคลุมมาตรฐาน/มาตรากฎหมายทั้งหมด ใช้ซ้อมความเข้าใจได้ แต่ควรหา past papers จริงมาฝึกเพิ่มก่อนสอบจริง

## เทคโนโลยี

Static site — HTML, CSS, Vanilla JS ล้วน ไม่มี build step, เปิด `index.html` ได้ตรงๆ หรือ clone แล้วเปิดผ่าน local server ก็ได้
